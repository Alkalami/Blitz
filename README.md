# Blitz

An eBook framework (CSS + template).

## Compile

`lessc blitz.less blitz.css`

### Roadmap

- Pre-release (v.0.95), debug
- Convert LESS to SASS
- Create Web Page
- Launch v.1
- LESS/SASS Plugins

### Support

- Readium
- iBooks
- Kobo
- Kindle (mobi7 + KF8)
- Google Play Books
- RMSDK (a.k.a. eBooks’ IE 6)
